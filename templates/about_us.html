Вот небольшая документация к API

Чтобы получить все экземпляры одного типа введите команду образца: /api/<название типа> Типы: users(пользователи), goods(товары), orders(заказы), comments(комментарии).
Чтобы получить один экземпляр введите: /api/<название типа>/<номер id>.
Если вы хотите добавить экземпляр, то помимо id необходимо будет ввести и другие параметры /api/<название типа>/<номер id>/<параметр 1>/<параметр 2>
